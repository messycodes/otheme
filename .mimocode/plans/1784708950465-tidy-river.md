# 修正模块检测路径

## 问题
`SuFileOperations.OTHEME_SYSTEM_DIR` 和 `LogExporter.OTHEME_MODULE_DIR` 路径不正确，模块检测应该检查 `/data/adb/modules/otheme/system_ext`。

## 修改

### 1. `SuFileOperations.kt:24`
```
- private const val OTHEME_SYSTEM_DIR = "/data/adb/modules/otheme/system"
+ private const val OTHEME_SYSTEM_DIR = "/data/adb/modules/otheme/system_ext"
```

### 2. `LogExporter.kt:25`
```
- private const val OTHEME_MODULE_DIR = "/data/adb/modules/otheme"
+ private const val OTHEME_MODULE_DIR = "/data/adb/modules/otheme/system_ext"
```

## 涉及文件
- `app/src/main/java/com/chuishui/otheme/SuFileOperations.kt`
- `app/src/main/java/com/chuishui/otheme/LogExporter.kt`

## 验证
- `SuFileOperations.checkOthemeDir()` 检查 `/data/adb/modules/otheme/system_ext` 是否存在
- `LogExporter.checkOthemeModule()` 检查同一路径
- `LogExporter` 目录检查段显示正确路径
